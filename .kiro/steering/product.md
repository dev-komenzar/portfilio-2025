# Portfolio Showcase

## Product Overview
This is a personal portfolio website designed to showcase works, skills, and contact information. The site is built with a focus on visual appeal, responsiveness, and multilingual support (Japanese and English).

## Key Features
- Work showcase with detailed work pages
- Skills section organized by categories with proficiency levels
- Responsive design optimized for all devices
- Smooth scrolling navigation with section highlighting
- Multilingual support (Japanese as base language, English as secondary)
- Dark mode by default with light mode option

## Target Audience
- Potential employers and clients
- Fellow developers and collaborators
- Anyone interested in the developer's work and skills

## User Experience Goals
- Present works in a visually appealing way
- Provide easy navigation between sections
- Ensure fast loading times and smooth interactions
- Support both Japanese and English-speaking visitors
